Documentation

Four Categories

Theme

We think that our app has an easily identifiable theme. It shoots procedurally generated fireworks based on user specifications. We think that we made it somewhat special and different, especially with the rocket and randomness we added in.

User Experience

We think that the app is relatively easy to use as everything is labeled. It is easy to identify what the program is and how it is used. The app also runs without errors.

Media

Everything passes validation, and we met all of the requirements for styling. We used the canvas API in an interesting way in order to draw fireworks using phyllotaxis. We used an external stylesheet and used an embedded font from Google Fonts.

Code

We followed the normal file naminng conventions and the coding standards. We used an external JavaScript file and both an external library and one that we made.

Wrong and Right

Right
